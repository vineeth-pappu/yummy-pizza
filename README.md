### Installation

Yummy Pizza requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ npm install -d
$ npm run start-dev
```

For production environments...

```sh
$ npm run build
$ npm start
```
